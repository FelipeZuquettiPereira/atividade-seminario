Conceito da propriedade display

A propriedade css display` define como um elemento aparece no fluxo do documento, afetando o layout, a visibilidade e a interação com outros elementos.
Ela determina se os elementos se comportam como `block`, `inline`, `flex`, `grid` ou outros tipos especializados



Explicação da propriedade position

A propriedade position permite definir a localização de elementos na página, e pode ser acompanhada por outras tais como : Top, Right, Bottom, left, que determinam como ficam a posição final do objeto. Mas também outros muito importantes são:

- static
Position static é o valor padrão no CSS, posicionando o elemento no fluxo normal da página conforme a ordem do HTM

- relative
O elemento é posicionado de acordo com o fluxo normal do documento e, em seguida, deslocado em relação a si mesmo com base nos valores de top, right, bottom, e left.
O deslocamento não afeta a posição de nenhum outro elemento; portanto, o espaço concedido ao elemento no layout da página é o mesmo que se a posição fosse static.

- absolute
O elemento é removido do fluxo normal do documento e nenhum espaço é criado para ele no layout da página. O elemento é posicionado em relação ao seu ancestral posicionado mais próximo (se houver) ou ao bloco que o contém inicialmente.
Sua posição final é determinada pelos valores de top, right, bottom, e left.

- fixed
funciona de forma diferente em relação às demais. Embora seja parecida com o posicionamento absoluto, com fixed o elemento “responde”, em termos de layout, ao navegador.
Isso significa que, mesmo que naveguemos na página para cima e para baixo, o elemento ficará fixo na mesma posição sempre.

- sticky
O elemento é posicionado de acordo com o fluxo normal do documento e, em seguida, deslocado em relação ao seu ancestral de rolagem mais próximo.



Conhecendo os display

- display: block
significa que o elemento é exibido como um bloco, assim como parágrafos e cabeçalhos sempre foram.

- display: inline
Organiza elementos como por exemplo imagens uma ao lado da outra em uma linha.

- display: inline-block O elemento vai ser inline, ou seja, ele vai ocupar no site apenas a sua largura e altura originais, então se tiver mais elementos, eles ficam um do lado do outro, porém não dá pra definir width(largura) nem height(altura) nem margin nem padding.
O block consegue fazer isso, ele ocupa 100% da largura, porém ele deixa você definir altura, largura, margin e padding, então este é o segredo do inline-block: Ele é inline, porém ele herda alguns comportamentos do block, e o principal é este de definir altura, largura etc.

- display: flex Ele é um jeito de organizar elementos um ao lado do outro.

- display: inline-flex
display: inline-flex: O contêiner é inline, ocupando apenas a largura do conteúdo (não quebra linha)

- display: grid
o display grid: ele cria um container e ajuda a organizar elementos em linhas em colunas, parecendo uma tabela.

- display: inline-grid
Em CSS, cria-se um contêiner de grade que atua como um elemento em linha, situado dentro de uma linha de texto ou ao lado de outros elementos em linha

- display: flow-root
Quando você tem elementos dentro de um container (div) com float: left ou right, o contêiner geralmente colapsa (fica com altura zero) porque não reconhece o tamanho dos filhos.
O flow-root força o contêiner a se expandir para abraçar todo o conteúdo flutuante, sem precisar de hacks adicionais.

- display: table
Uma tabela em CSS é definida como um elemento de estrutura de dados composto por linhas e colunas para dados, para cabeçalhos que organiza informações de forma visual em uma grade.
Com CSS, essa estrutura pode ser estilizada para melhorar a legibilidade, alinhamento e aparência visual de grandes volumes de dados, produtos ou preços.

- display: block flex
serve para definir o comportamento externo e interno de um contêiner: ele ocupa toda a largura da linha (block) enquanto organiza seus itens filhos de forma flexível e alinhada

- display: block grid
O elemento se comporta como um elemento de bloco e apresenta seu conteúdo.

- display: block flow-root
serve para criar um novo contexto de formatação de bloco no elemento, garantindo que ele se comporte como um bloco (display: block) enquanto impede que flutuações internas afetem o layout externo ou que margens superiores/inferiores dos filhos ultrapassem as bordas do pai.

- display: inline flex
é uma propriedade CSS que transforma um elemento em um contêiner flexível, mas o mantém na linha (inline) com o conteúdo ao redor, em vez de pular para uma nova linha.

- display: inline grid
O display: inline-grid no CSS transforma um elemento em um contêiner de grade (grid), mas o faz com comportamento inline em vez de bloco. Isso significa que o contêiner não ocupa toda a largura da linha.

- display: inline flow-root
é uma combinação CSS que faz um elemento se comportar como uma caixa em linha (inline), alinhando-se ao fluxo de texto, ao mesmo tempo em que cria um novo contexto de formatação de bloco.
